# Codility Training problems

### Tricks
- pivots
- hashsets
- lazy updates
- prefix sum
    - rolling sum
    - next occurence

### Edge cases
1. Empty arrays
2. Modulo return values
3. Failure return values like -1
4. Return -1 for values larger than threshold
5. Check Loop bounds (0 -> n) or (1 -> n + 1)
